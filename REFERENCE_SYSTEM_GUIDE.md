# Intelligent Reference Generation System
## Complete Guide to Smart Bibliography

---

## How It Works

When a teacher creates a session plan, the system:

1. **Analyzes** the topic, module title, and learning outcomes
2. **Identifies** the subject matter (Programming, Database, Networking, etc.)
3. **Selects** 4-5 highly relevant, authoritative references
4. **Formats** them in proper APA 7 format
5. **Automatically** includes them in the Bibliography section

**All without any teacher intervention!**

## Reference Format Details

### APA 7 Format Structure

All references follow this format:

```
Author/Organization. (Year). Title of work. Publisher.
```

**Examples**:

**Book with single author**:
> Downey, A. (2015). Think Python: How to think like a computer scientist. O'Reilly Media.

**Book with two authors**:
> Deitel, P., & Deitel, H. (2019). Python for programmers. Pearson Education.

**Book with multiple authors**:
> Garcia-Molina, H., Ullman, J. D., & Widom, J. (2008). Database systems: The complete book (2nd ed.). Prentice Hall.

**Organization as author**:
> Rwanda Education Board. (2021). TVET curriculum framework. REB Publications.

**Online resource**:
> Mozilla Foundation. (2023). Web development documentation and standards. Retrieved from https://developer.mozilla.org
